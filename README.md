# 🏪 店點評估 API (v28 穩定版)

這是一個基於 FastAPI 的 API，用於計算位點開發潛力與營運預測。運用Vibe Coding製作。

### ✨ 主要功能
* **模型預測**：回傳營運推薦分數。
* **SHAP 自動計算**：提供影響分數的前兩大核心因素，回傳所有特徵shap值。
* **業態自動校準**：針對超市 (1.58) 與超商 (0.13) 設定不同的基準。

### 🚀 快速啟動
1. 安裝環境：`pip install -r requirements.txt`
2. 啟動服務：`uvicorn main:app --reload`

### 📊 API 回傳格式範例
成功推論後會回傳如下結構，包含位點評分與 SHAP 影響力分析：
- `score`: 綜合評分 (0-100)
- `report`: 展店建議
- `shap_values`: 各項特徵對分數的貢獻度
